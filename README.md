# Road Accident Data Analysis & Safety Insights

## Project Overview
This project analyzes road accident data to identify patterns and factors that influence accident severity and frequency. The goal is to understand how variables such as weather conditions, vehicle types, and location impact road safety.

An interactive dashboard was created using **Tableau** to visually explore accident trends and support data-driven decision making.

---

## Objective
The main objectives of this project are:

- Analyze accident severity under different **weather conditions**
- Study the relationship between **vehicle type and accident severity**
- Identify **high-risk areas** in urban and rural locations
- Provide **data-driven insights** that can help improve road safety

---

## Dataset Description
The dataset contains information related to road accidents, including:

- Accident severity  
- Weather conditions  
- Road surface conditions  
- Vehicle types involved  
- Urban vs rural accident locations  
- Other contributing factors

These attributes help analyze how environmental and infrastructure factors influence accidents.

The analysis focuses on questions such as:
